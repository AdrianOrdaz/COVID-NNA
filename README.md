# COVID-19 NNA
Este proyecto consiste en la evaluación de 4 modelos de redes neuronales artificiales sobre la predicción de muertes a causa del COVID-19. Se cuenta con un dataset del gobierno mexicano que contiene registros de 20,000 personas que contrajeron dicha enfermedad (cuya mitad falleció) y de las cuales se tiene un total de 24 características de las cuales se apoyarán los modelos para encontrar patrones e identificar si existe relación entre las características (padecimientos) de la persona y fallecer de COVID-19. 
Estas predicciones se hicieron en fundamento con el concepto de clasificación, en redes neuronales de sklearn, existe la técnica MLPClassifier, la cual se uso, y que, consiste en el método de entrenamiento de perceptrones multicapa.
